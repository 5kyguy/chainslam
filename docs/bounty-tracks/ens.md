# ENS

**Prize Pool:** $5,000

## About

ENS is the identity layer for the new internet. It turns wallet addresses into human-readable names like `yourname.eth` — a portable onchain profile that works across apps, chains, and wallets.

Developers use ENS to replace raw addresses with real identities, build decentralized websites, and create scalable subname ecosystems on L2. As AI agents become first-class onchain actors, ENS gives them names, reputations, and discoverability.

Used by Coinbase, Uniswap, Venmo, and many others, ENS is the username layer for web3.
