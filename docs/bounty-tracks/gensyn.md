# Gensyn

**Prize Pool:** $5,000

## About

Gensyn is building open infrastructure for AI.

AXL is a peer-to-peer network node: a single binary that provides encrypted, decentralized communication for applications. Apps talk to `localhost`, while AXL handles encryption, routing, and peer discovery across the mesh.

AXL includes built-in MCP and A2A support for structured agent-to-agent communication, with end-to-end encryption by default. Any language that can make HTTP requests can use it.

For this hackathon, Gensyn is looking for builders who want to push what is possible when AI agents communicate peer-to-peer with no central coordinator.

## Judging Criteria

Projects are evaluated on:

- Depth of AXL integration
- Code quality
- Clear documentation
- Working examples

## Qualification Requirements

- Must use AXL for inter-agent or inter-node communication (no centralized message broker replacing AXL)
- Must demonstrate communication across separate AXL nodes, not just in-process
- Must be built during the hackathon
