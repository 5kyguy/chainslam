# Uniswap Foundation

**Prize Pool:** $5,000

## About

The Uniswap Protocol is one of the largest decentralized exchange protocols for swapping value onchain across Ethereum and other supported blockchains.

The Uniswap ecosystem has evolved into a full-stack platform for onchain finance, including smart contracts such as v2, v3, and v4, developer tools, APIs, and emerging infrastructure like Unichain. This enables developers to build applications on top of shared, permissionless liquidity.
